# copy 复制

::: tip
`@netang/quasar/utils/copy.js` [Github](https://github.com/netangsoft/netang-quasar/blob/main/utils/copy.js) [Gitee](https://gitee.com/jinmarcus/netang-quasar/blob/main/utils/copy.js)
:::

- 示例

```javascript

// 复制并提示：复制成功
copy('test1')

// 复制并提示：复制【test1】成功
copy('test1', true)

// 复制并提示：ok
copy('test1', 'ok')

// 复制并关闭自动提示
copy('test1', false)
```
