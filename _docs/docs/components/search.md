# Search 搜索

::: tip
搜索组件
:::

| 属性名             | 说明      | 类型         | 可选值 | 默认值 |
|-----------------|---------|------------|-----|-----|
| value / v-model | 绑定值（必填） | -          | -   | -   |
| options         | 参数      | `Array`    | -   | -   |
| onSearch        | 搜索      | `Function` | -   | -   |
| onReset         | 重置      | `Function` | -   | -   |
