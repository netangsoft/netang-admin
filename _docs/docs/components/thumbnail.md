# Thumbnail 缩略图

::: tip
继承 [`<n-img>`](./img.md) 所有特性
:::

| 属性名     | 说明       | 类型                            | 可选值 | 默认值   |
|---------|----------|-------------------------------|-----|-------|
| size    | 图片尺寸     | `Number`                      | -   | 40    |


示例

```javascript
<n-thumbnail
    src="..."
/>
```
