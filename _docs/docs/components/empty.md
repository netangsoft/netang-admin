# Empty 空状态

::: tip
空状态
:::

| 属性名         | 说明   | 类型                  | 可选值 | 默认值  |
|-------------|------|---------------------|-----|------|
| icon        | 图标   | `String`            | -   | info |
| icon-size   | 图标大小 | `String` / `Number` | -   | 70   |
| description | 描述文字 | `String`            | -   | -    |
