# ColumnTitle 栏目标题

::: tip
**栏目标题**

用于显示一个撑满一行的栏目标题
:::

## 使用示例

### 基础示例

```html
<!-- 栏目标题 -->
<n-column-title label="订单信息" />
```

## ColumnTitle API

### ColumnTitle 属性

| 属性名     | 说明   | 类型                  | 可选值 | 默认值 |
|---------|------|---------------------|-----|-----|
| label   | 标签   | `String` / `Number` | -   | -   |
| tooltip | 提示信息 | `String` / `Number` | -   | -   |
