# EditorCode 代码编辑器

::: tip
代码编辑器
:::

| 属性名             | 说明       | 类型                  | 可选值 | 默认值   |
|-----------------|----------|---------------------|-----|-------|
| value / v-model | 绑定值（必填）  | -                   | -   | -     |
| width           | 宽度       | `String` / `Number` | -   | 100%  |
| height          | 高度       | `String` / `Number` | -   | -     |
| language        | 脚本语言     | `String`            | -   | -     |
| readonly        | 是否只读     | `Boolean`           | -   | false |
| minimap         | 是否显示代码视图 | `Boolean`           | -   | false |
| tab-size        | tab 长度   | `Number`            | -   | 4     |
| theme           | 主题       | `String`            | -   | vs    |
